# TaskFlow — SaaS Project & Task Management

A full-stack SaaS application for team project and task management.

## Architecture

- **Frontend**: React + Vite + TailwindCSS (port 5000)
- **Backend**: Node.js + Express REST API (port 3001)
- **Database**: PostgreSQL (Replit managed)

## Features

- Secure JWT authentication (register/login)
- Workspaces with role-based access (Admin, Manager, Member)
- Projects with color coding, due dates, progress tracking
- Tasks with priorities, assignees, labels, comments
- Kanban board with drag-and-drop
- Dashboard analytics with activity feeds
- Notifications system
- Settings page (profile, password, notifications)

## Development

Start everything with the "Start application" workflow, which runs `start.sh`.

- Frontend dev server: http://localhost:5000
- Backend API: http://localhost:3001/api

## User Preferences

- Dark theme only
- Concise, information-dense UI
- Professional SaaS design

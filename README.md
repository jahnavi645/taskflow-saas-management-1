# TaskFlow – SaaS Project Management System

## 📌 Project Overview

TaskFlow is a full-stack SaaS-based project management application developed to help individuals and teams organize projects, manage tasks, collaborate with members, and track work progress efficiently.

The application provides a simple and user-friendly interface for managing projects with secure user authentication and real-time data storage.

---

## ✨ Features

### 🔐 User Authentication
- User Registration
- Secure Login and Logout
- Session Management

### 📂 Project Management
- Create Projects
- Edit Project Details
- Delete Projects
- Manage Multiple Projects

### ✅ Task Management
- Create Tasks
- Update Task Status (Pending, In Progress, Completed)
- Delete Tasks
- Track Project Progress

### 👥 Team Management
- Add Team Members
- Collaborate with Team Members
- Manage Workspace Activities

### 📊 Dashboard & Reports
- View Total Projects
- View Total Tasks
- Track Completed Tasks
- Monitor Overall Progress
